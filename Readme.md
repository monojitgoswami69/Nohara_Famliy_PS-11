## 🧠 About The Project

**CodeCollab** is a real-time collaborative coding platform where multiple users can write and edit code together instantly.

---

## ✨ Features

- ⚡ Real-time code sync using WebSockets  
- 🎨 Modern UI (Tailwind CSS)  
- 🚀 Fast React + Vite frontend  
- 🔌 Live collaboration rooms  
- 🧠 Python backend APIs

## 🚀 Getting Started

### Clone Repo

```bash
git clone https://github.com/your-username/Nohara_Family_PS-11.git
cd Nohara_Family_PS-11
```

---

### Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

### Socket Server

```bash
cd backend-socket
npm install
node server.js
```

---

## 🔐 Environment Variables

```env
PORT=5000
API_URL=http://localhost:5000
SOCKET_URL=http://localhost:3001
```
